## About this Project

![Sky Guide Screen Shot](https://i.imgur.com/hu9CA2Y.png)

That is a ton of weather apps available, but I wanted to take a crack at it for something that really suits my needs. 

Here's why:
* This app combines the use of Multiple API in a seamless PWA
* Built with internationalization in mind
* Open source :)

Of course this app might not serve every possible function, but I will be adding more features in the near future. You may also suggest changes by forking this repo and creating a pull request or opening an issue.


### Built With

* [React](https://reactjs.org/)
* [Ant Design](https://ant.design/)
* [i18next](https://www.i18next.com/)
* [Less](http://lesscss.org/)

## Getting Started

Insturctions to set up your project locally. Follow these instruction to get a local copy up and running


### Prerequisites

* npm
```sh
npm install npm@latest -g
```


### Installtion

1. Clone the repo
```sh
git clone https://github.com/majicmaj/skyguide.git
```
2. Install NPM packages
```sh
npm install
```
3. Run the server
```sh
npm start
```


## Usage

You can run this project locally, just make sure to change the `dev` enviroment to something else in `constants.js`
```JS
export const ENV = {
  DEV: 'development' //change this to something else.
  ...
}
```
